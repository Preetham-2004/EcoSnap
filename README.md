## 🌍 Problem Statement  
Improper waste segregation leads to **recycling inefficiencies, increased landfill waste, and environmental harm**.  
To tackle this, we have developed **EcoSnap**—an **AI-powered waste classification tool** that helps users **identify and categorize waste efficiently**.  

🌱 **EcoSnap promotes sustainable habits by:**  
✅ **Using image recognition & speech-to-text** to classify waste as **Recyclable, Non-Recyclable, or Biodegradable**  
✅ **Providing disposal guidance** based on **location-specific recycling rules**  
✅ **Encouraging user participation** through **gamification and reward points**  

---

## 💡 Solution - EcoSnap  
EcoSnap is a **web-based AI-powered waste classification tool** that enables users to:  
📸 **Upload an image** of waste or **describe the waste using voice input**.  
🎙️ **Speech-to-Text Processing** to classify waste based on user descriptions.  
🤖 **Get AI-based classification** (Recyclable, Non-Recyclable, or Biodegradable).  
🏆 **Earn points** for correctly sorting waste.  
🗑️ **Receive disposal guidance** based on location-specific recycling rules.  

---

## 🔄 Workflow / How It Works  
1️⃣ **User uploads an image of waste or speaks about the waste** (Speech-to-Text).  
2️⃣ **AI model classifies** it into **Recyclable, Non-Recyclable, or Biodegradable**.  
3️⃣ **Gamification system** rewards points for correct sorting.  
4️⃣ **Disposal instructions** are provided based on classification.  
5️⃣ **Location-based recycling guidelines** are displayed for better decision-making.
