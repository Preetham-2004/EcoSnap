# 🚀 EcoSnap – AI-Powered Waste Classification

## 🌍 Overview
EcoSnap is a web-based AI solution that helps users **identify and properly dispose of waste** using image recognition and voice input. It promotes sustainable habits by simplifying waste segregation into:
- ♻️ Recyclable  
- 🚫 Non-Recyclable  
- 🌱 Biodegradable  

---

## 💡 Problem
Improper waste segregation leads to:
- Low recycling efficiency  
- Increased landfill waste  
- Environmental damage  

---

## 🧠 Solution
EcoSnap uses AI to:
- 📸 Classify waste from images  
- 🎙️ Process voice input (Speech-to-Text)  
- 🗑️ Provide disposal guidance  
- 🏆 Encourage usage through gamification  

---

## 🔄 How It Works
1. Upload image or describe waste via voice  
2. AI classifies the waste  
3. Get disposal instructions  
4. Earn points for correct sorting  

---

## ✨ Key Features
- AI-based Image + Voice Classification  
- Real-time Disposal Guidance  
- Gamification (points & engagement)  
- Location-aware recommendations  
- Simple and intuitive UI  

---

## 🛠️ Tech Stack
**Frontend:** React (Vite), TypeScript, Tailwind CSS  
**Backend:** FastAPI  
**AI Models:**  
- BERT (NLP & text classification)  
- MobileNetV2 (image classification)  

---

## 🌱 Impact
- Improves waste segregation  
- Promotes sustainable behavior  
- Supports smart city initiatives
